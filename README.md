## 🚀 Cómo usar la aplicación

1. Accede a [https://knowme.jordivu.es](https://knowme.jordivu.es)
2. Inicia sesión como administrador con las credenciales:
   - **Email:** `admin@admin.com`
   - **Contraseña:** `damdaw`

### Panel de usuarios
- Ve a la pestaña **"Usuarios"** para editar, eliminar, crear y notificar a los usuarios.

### Generar una invitación
- Entra en tu **perfil** y pulsa el botón **"Generar"**.

### Crear un portfolio
- Pulsa el botón **"Nuevo portfolio"** en la página de portfolios.
